Website

Website adalah kumpulan halaman web yang dapat diakses melalui internet. Setiap halaman web dapat berisi teks, gambar, video, dan berbagai konten lainnya. Website digunakan untuk menyediakan informasi, layanan, komunikasi, dan banyak tujuan lainnya. Mereka dapat digunakan untuk publikasi pribadi, bisnis, pendidikan, atau hiburan, dan dapat diakses melalui browser web di perangkat komputer atau perangkat seluler. Website biasanya memiliki alamat unik yang disebut URL (Uniform Resource Locator) yang memungkinkan pengguna untuk mengaksesnya.
